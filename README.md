# MusicPlayerTK
A MusicPlayer made in Python with Tkinter

# Requirements
Pygame:
  ```
  pip install pygame
  ```

Cairosvg:

  ```
  pip install cairosvg
  ```
Weasyprint and GTK libraries for Cairosvg:

  ```
  https://weasyprint.readthedocs.io/en/latest/install.html
  ```
  
Eyed3:

  ```
  pip install eyed3
  ```
